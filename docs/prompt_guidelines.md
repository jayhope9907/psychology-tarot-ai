# Prompt guidelines (wellness education — not medical practice)

This repository's live system prompts and legal layer define the product as
**AI wellness / self-reflection / association training aid**, not clinical care.

- Do not instruct the model to diagnose, prescribe, or replace licensed clinicians.
- Crisis: direct to 1393 / 119 / 129 / 1577-0199.
- Prefer the runtime builders in `app/prompt_config.py` and
  `app/services/legal_compliance.py` over outdated narrative language.

See also: `docs/ip/invention-disclosure.md`, `/innovation`, `/legal`.
